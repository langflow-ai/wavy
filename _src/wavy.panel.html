set_training_split(test_size=0.1, val_split=0.2)[source]

Time series split into training, validation, and test sets, avoiding data leakage. Splits the panel in training, validation, and test panels, accessed with the properties .train, .val and .test. The sum of the three sizes inserted must equals one. :param val_size: Percentage of data used for the validation set. :type val_size: float :param test_size: Percentage of data used for the test set. :type test_size: float

Example: >>> panel.set_training_split(val_size=0.2, test_size=0.1) >>> train = panel.train >>> val = panel.val >>> test = panel.test
